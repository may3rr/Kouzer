# 抠字儿

一个极简的 AI 论文、长文本校对工具。粘贴文本，静候 AI 逐行抠出语病与错别字。

## 使用方式
1. 打开 [在线页面](https://jackielyu.is-a.dev/Kouzer/)
2. 点击「配置」，填入你的 API Key
3. 从 Word 中复制**纯文本**内容，粘贴至输入框
4. 点击「开始校对」

## 局限
1. 输入框只支持纯文本。论文中的数学公式强行粘贴会产生字符丢失，容易导致 AI 误判。
2. AI 是助理，不是法官。校对结果请视为**参考建议**，自行甄别。
